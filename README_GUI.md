# FolderRAG Desktop 🚀

## 快速开始

### 方式一：双击运行（推荐）
直接双击 **`启动 FolderRAG Desktop.bat`** 即可一键启动！

首次运行会自动：
1. 创建 Python 虚拟环境
2. 安装所需依赖
3. 启动图形界面

### 方式二：命令行启动
```bash
python gui.py
```

### 方式三：从命令行打开指定文件夹
```bash
python gui.py "C:\你的文件夹路径"
```

---

## 功能特性

### 📂 文件夹选择
- **点击浏览**：点击蓝色区域或「浏览文件夹」按钮
- **拖拽支持**：如果安装了 `tkinterdnd2`，支持直接拖入文件夹
- **快捷键**：`Ctrl + O` 打开文件夹

### 📋 文件列表
- 自动扫描文件夹中所有文件
- 显示文件名、大小、是否支持解析
- 支持与不支持的文档一目了然

### ⚙️ 输出设置
| 设置项 | 说明 |
|--------|------|
| **输出路径** | 生成的 HTML 保存位置 |
| **HTML 文件名** | 自定义输出文件名 |
| **最大字符数** | 控制输出总字符数（默认 50000），留空=不限 |
| **包含不支持文件** | 勾选后在 HTML 中标记不支持的文件 |

### 🚀 一键生成
- 点击「一键生成 HTML」开始解析
- 进度条实时显示
- 完成后弹出详细报告
- 可选择打开文件所在文件夹

### ⌨️ 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + O` | 打开文件夹 |
| `Ctrl + G` | 生成 HTML |
| `Ctrl + S` | 选择输出路径 |
| `Esc` | 退出程序 |

---

## 输出 HTML 特点

生成的 HTML 文件包含：
- ✅ 文件头部汇总（文件名、字符数、导出时间）
- ✅ 每个文件独立 `<article>` 区块
- ✅ 文件大小和内容字符数标注
- ✅ 不支持的文档自动标记 `⏭️`
- ✅ 现代简洁 UI 样式，浏览器直接打开即可阅读
- ✅ 超出字符限制自动截断并提示

## 系统要求

- **操作系统**：Windows / macOS / Linux
- **Python**：3.8 及以上版本
- **依赖**：首次运行自动安装（`requirements.txt`）

## 技术栈

- **UI 框架**：tkinter + ttk（Python 标准库）
- **文档解析**：pdfminer.six / python-docx / python-pptx / openpyxl
- **拖放支持**：tkinterdnd2（可选安装）
